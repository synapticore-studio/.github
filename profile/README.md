## Focus Areas

- **DCC Pipelines** — Blender, Houdini, Maya, USD workflows with AI automation
- **Game Development** — Godot 4.6 plugins, web exports, GPU particle systems
- **Virtual Production** — Projection mapping, camera tracking, color science
- **3D & Rendering** — PBR engines, neural rendering, procedural generation
- **Unreal Engine** — OpenVINO inference, OpenStreetMap integration, AMUM framework
